# Clinique Massotherapie West

Single-page bilingual website (French/English) for Clinique Massotherapie West.

## Structure

- `/en/` - English version
- `/fr/` - French version
- `/assets/` - Shared assets (images, CSS, JS)
- `_redirects` - Cloudflare Pages routing configuration

## Deployment

This site is configured for Cloudflare Pages deployment.
